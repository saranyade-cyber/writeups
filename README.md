# My Cybersecurity Sprint

# writeups
Lab writeups for PortSwigger Web Security Academy

### Why I started it
I started this cybersecurity sprint to transition from theoretical computer science concepts
to practical, hands-on vulnerability analysis. By looking into real-world laboratory
environments, I wanted to understand exactly how web applications are broken into
and more importantly, how to secure them against modern attack vectors.
